# Asian Paints Financial Model

An integrated 3-Statement Financial Model of Asian Paints Ltd. built in Microsoft Excel for educational purposes.

## Key Features

- Integrated 3-Statement Financial Model
- DCF Valuation
- Relative Valuation
- WACC Calculation
- Revenue & Expense Forecasting
- Working Capital, CapEx & Depreciation Schedules
- Scenario Analysis (Base, Best & Worst Case)
- Interactive Dashboard
